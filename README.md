# I ❤️ JSON

## Prerequisites
- NodeJS v16.x
- Visual Studio Code

## Getting Started

Installing Dependencies
```bash
yarn
```

Run Server

```bash
yarn dev
```

Or start with Docker

```bash
docker-compose up
```

## API Sample
http://localhost:3000/api/ilovejson
